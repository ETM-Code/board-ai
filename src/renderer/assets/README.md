# Wiley house music

While Wiley is working on the board, background music plays (toggle it with the
note button next to the microphone). The music fades out whenever you or Wiley
are speaking and fades back in when the conversation pauses.

To use "Wearing My Rolex" by Wiley as the track, drop your own copy of the
audio file into this directory named:

```
wearing-my-rolex.mp3   (or .m4a / .ogg / .wav)
```

The build picks it up automatically and loops it. No audio ships with the
repository — you must supply a copy you have the rights to play. When no file
is present, a synthesized house loop plays instead.
