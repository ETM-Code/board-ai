# board-ai

Here are the tools that we’ll use:

  
Voice Model: openai-realtime-2.1 ([https://developers.openai.com/api/docs/models/gpt-realtime-2.1](https://developers.openai.com/api/docs/models/gpt-realtime-2.1))  
Agent Harness: pi agent ([https://github.com/earendil-works/pi](https://github.com/earendil-works/pi))  
Whiteboard: Excalidraw ([https://github.com/excalidraw/excalidraw](https://github.com/excalidraw/excalidraw))  
Bundling: Electron ([https://www.electronjs.org/](https://www.electronjs.org/))

Only bit of frontend that we need beyond excalidraw should be a mute/unmute button in the bottom right hand corner. As this is an agent with auto-approve or something similar, it can just do whatever on the computer anyway, so that should be all good.
